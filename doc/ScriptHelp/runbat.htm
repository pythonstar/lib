runbat

该函数创建一个临时BAT文件并执行

boolean runbat(
  string bat,
  string temppath
)

参数

string
将要被执行的BAT内容
string
BAT文件创建的临时目录，BAT文件执行完成后临时文件会自动被删除。该参数可省略，默认为应用程序启动的目录。

返回值

boolean
成功返回true，失败返回false

示例

require 'star'
success = star.runbat(
	[[
	notepad
	pause
	calc
	]])
