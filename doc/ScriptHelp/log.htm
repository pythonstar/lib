log

该函数将字符串信息输出到log文件中

log(string, [string logfilename], [boolean append])

参数

string
字符串类型或者可以转换为字符串的类型
string
log文件名或全路经，可选，如果为nil则默认使用当前脚本的目录下log.txt日志文件
boolean
是否追加的方式输出，默认为false

返回值

boolean
日志内容输出成功则为true，否则为false

示例

require 'star'

star.log('test')

star.log('test',nil,true)

star.log('test','c:/log.txt',true)

star.log('test','log.txt',true)
