downloadfile

该函数通过调用URLDownloadToFile下载指定网址文件到本地文件中

boolean, string downloadfile(
  string url
  string filepath
)

参数

string
网址
string
本地文件路径

返回值

boolean
true表示成功，false表示失败
string
成功返回“ok”，失败返回表示失败原因的字符串

示例

require 'star'
success, msg = star.downloadfile('www.baidu.com', [[e:\test.html]])
