reversefind

该函数通过调用CString::ReverseFind()倒叙查找字符串中某字符出现的位置

number reversefind(
  string s1
  string s2
)

参数

string
字符串，主串
string
字符串，子串，如果字符串较长，仅取第一个字符作为查找对象

返回值

number
返回字符串首字符在主串中出现的最后一个位置，序号以Lua标准从1开始，如果没找到则返回null

示例

require 'star'

pos = star.reversefind(' hello ', 'l')
