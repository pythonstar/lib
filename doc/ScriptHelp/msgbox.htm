msgbox

该函数用以弹出系统对话框

number msgbox(
	var text,
	var caption,
	number utype
)

参数

var
任何类型，将被转换为字符串后作为对话框内容显示
var
任何类型，将被转换为字符串后作为对话框标题显示，该参数可省略，默认为：提示
number
对话框显示显示方式，意义如下。该参数可省略，默认为MB_OK
