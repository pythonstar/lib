loadlibrary

该函数调用GetModuleHandle、LoadLibrary获取一个动态库的句柄

number loadlibrary(
  string name
)

参数

string
动态库的名称

返回值

number
动态库的句柄，成功返回非0，失败返回0

示例

require 'star'

user32 = star.loadlibrary('user32.dll')
if user32~= 0 then 
	MessageBox = star.getprocaddress(user32, 'MessageBoxA')
	if MessageBox~=0 then 
		MessageBox(0, 'hello', '', 0)
	end
end
