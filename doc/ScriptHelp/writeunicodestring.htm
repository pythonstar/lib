writeunicodestring

创建一个UNICODE文件并写入文本，UNICODE文件起始字符为：0xFF 0xFE

boolean writeunicodestring(
  string filename,
  string text
)

参数

string
文件全路经
string
要写入文件的文本

返回值

boolean
成功返回true，失败返回false

示例

require 'star'

success = star.writeunicodestring([[c:\1.txt]], 'hello')
