filterinvalidfilename

该函数过滤掉Windows下的非法文件夹名字符，如：\/:*?'"<>|，并去掉字符串末尾的.

string filterinvalidfilename(
  string filename
)

参数

string
字符串

返回值

string
返回过滤处理后的字符串。空串返回''，不返回null。

示例

require 'star'
s = star.filterinvalidfilename([[:新建文件?夹...]])
