unescapexml

该函数反转义html、xml中的转义字符

string unescapexml(
  string s
)

参数

string
html、xml字符串

返回值

string
返回反转义后的字符串，反转义规则如下。空串返回''，不返回null。
	strText.Replace(_T("&"),_T("&"));
	strText.Replace(_T("<"),_T("<"));
	strText.Replace(_T(">"),_T(">"));
	strText.Replace(_T("""),_T("\""));
	strText.Replace(_T("'"),_T("\'"));
	strText.Replace(_T(" "),_T(" "));
	strText.Replace(_T("©"),_T(" "));
	strText.Replace(_T("®"),_T(" "));
	strText.Replace(_T("/"),_T("/"));
	strText.Replace(_T("<"),_T("<"));
	strText.Replace(_T(">"),_T(">"));
	strText.Replace(_T("&"),_T("&"));

示例

require 'star'
s = star.unescapexml('"hello"')
