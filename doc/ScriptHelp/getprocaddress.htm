getprocaddress

该函数调用GetProcAddressy获取导出函数的地址

number getprocaddress(
	number handle,
  string name
)

参数

number
动态库的句柄，通过loadlibrary获得
string
导出函数的名称

返回值

number
导出函数的地址，成功返回非0，失败返回0

示例

require 'star'

user32 = star.loadlibrary('user32.dll')
if user32~= 0 then 
	MessageBox = star.getprocaddress(user32, 'MessageBoxA')
	if MessageBox~=0 then 
		MessageBox(0, 'hello', '', 0)
	end
end
