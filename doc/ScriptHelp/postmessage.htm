postmessage

该函数调用PostMessage向窗口发送消息

boolean postmessage(
  number hwnd,
  number msg,
  var wparam,
  var lparam
)

参数

number
窗口句柄
number
消息
var
消息参数，可以为0，nil，数值，字符串，userdata
var
消息参数，可以为0，nil，数值，字符串，userdata

返回值

boolean
成功返回true，失败返回false

示例

require 'star'

local WM_CLOSE = 0x0010

success = star.postmessage(0x2345, WM_CLOSE)
